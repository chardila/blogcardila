## How to Use

Simply add the component to your MDX posts with a TMDB movie ID:

```mdx
import MovieTrailerCard from '@/components/MovieTrailerCard.astro';

<MovieTrailerCard movieId="550" />
```

### Finding Movie IDs

1. Go to [themoviedb.org](https://www.themoviedb.org/)
2. Search for your movie
3. The ID is in the URL

For example: `https://www.themoviedb.org/movie/550-fight-club` → Movie ID is `550`

## Features

- ✅ Embedded YouTube trailers
- ✅ Local caching (24 hour TTL)
- ✅ Spanish language support with English fallback
- ✅ Rich movie metadata (genres, ratings, synopsis)
- ✅ Fallback to backdrop/poster if no trailer available
- ✅ Theme-aware styling (light/dark mode)
